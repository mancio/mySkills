# mySkills

This repository contains my skills projects.

## Projects

- [llm-wiki](llm-wiki/SKILL.md) - a skill for bootstrapping an Obsidian-like LLM wiki (knowledge graph + Flask viewer + ingestion workflow) inside any project, or ingesting raw source material into curated Markdown pages.
- [multi-meteo](multi-meteo/SKILL.md) - a skill for building a local Flask weather app that combines multiple weather sources into an ensemble forecast with month, day, and hourly views.
- [stock-market](stock-market/SKILL.md) - a skill for building a stock support report app that finds US and European stocks near technical support and generates a sortable HTML report.
- [tax-advisor](tax-advisor/SKILL.md) - a skill for building a Python app that acts as a personal tax advisor specialized in country-specific tax law, researches questions live on the internet, and generates PDF reports with risk assessments.
